# 📊 Transaction & Customer Behavior Analysis (1M+ Records)

🚀 **End-to-End Data Analytics Project | Excel + ETL + Business Insights**

This project analyzes a large-scale transactional dataset (1M+ records) to uncover **customer behavior, payment patterns, and revenue trends** across **7,005 cities**. The objective is to transform raw data into **actionable business insights** using Advanced Excel and ETL techniques.

---

## 📌 Project Objective

To answer key business questions:
- How are customers transacting (online vs offline)?
- What are the monthly revenue trends?
- How does performance vary across cities?
- Which customers contribute the most value (CLV)?

---

## 🛠️ Tools & Technologies

- Advanced Excel  
- Pivot Tables & Charts  
- Power Query (ETL Pipeline)  
- Data Cleaning & Transformation

- Extracted raw transactional dataset (1M+ records)

## 🔍 Key Insights

### 💳 Payment Behavior
- **89% transactions via swipe**
- **11% transactions online**
➡️ Strong dependency on offline methods → opportunity for digital adoption  

---

### 📅 Revenue Trends
- Stable growth observed between **May–August**
- Peak performance in **August (~$4.68M)**
- Fluctuations across months indicate moderate stability  
- **November drop (-81.69%)** → potential anomaly or operational issue  

---

### 🌍 Geographic Insights
- Data spans **7,005 cities**
- High variability in transaction volumes across locations  
- Example: city-level spikes (e.g., July peak) followed by fluctuations  
➡️ Indicates need for **city-specific strategies**  

---

### 💰 Customer Lifetime Value (CLV)
- Small group of customers contributes majority of revenue  
➡️ Strong case for **retention-focused strategies**  

---

## 💡 Business Recommendations

- 🚀 Promote **digital payments** (cashbacks, incentives, UX improvements)  
- 📈 Align campaigns with **peak months (May–Aug)**  
- 🔍 Investigate **November anomaly**  
- 🎯 Implement **geo-targeted marketing strategies**  
- 🔁 Focus on **customer retention & loyalty programs**
